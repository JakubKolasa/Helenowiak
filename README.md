# Helenowiak
Projekt zakłada stworzenie robota od podstaw. Logika robota będzie oparta na mikrokontrolerze STM32. Zostanie on umieszczony na zaprojektowanej w programie KiCad płytce PCB. Model robota został zaprojektowany w programie AutoCAD Inventor.  Do detekcji przeciwnika wykorzystane zostaną czujniki zbliżeniowe działające w zakresie podczerwieni. W celu zastosowania bardziej skomplikowanych algorytmów walki i umożliwienia swobodnego poruszania się po dojo, robot zostanie wyposażony w czujniki odbiciowe mające na celu wykryć krawędzie areny, oraz enkodery umożliwiające dokładny pomiar przebytej odległości. Robot będzie komunikował się z komputerem poprzez moduł bluetooth dzięki czemu będzie można sterować nim w sposób zdalny przy pomocy odpowiednich komend.
